Как сделать команду /bug?

На этой стр​анице:

    Всем привет! Решил как-то раз я сделать команду для сообщения о баге в личные сообщения. Потому что без этой команды очень трудно передать баг разработчику. В этом гайде я решил показать как это сделать, чтобы Вы не тратили много времени на сложный код.

1. Введение. 2. Функции.

    Итак, что нам для этого надо? Для команды нам надо создать файлы reports.txt и report_num.txt в папке нашего бота, далее создаём 3 функции, первая которая будет считывать количество жалоб, ​делаем это для того, чтобы после каждого перезапуска, количество жалоб не приводилось заново к нулю, вторая функция создана для добавления 1 единицы жалобы, как только пользователь отправил данную команду, а третья функция будет сохранять сообщения, номер жалобы и дискорд отправителя в файле reports.txt. Пример кода Вы можете увидеть ниже.

def get_current_report_number ():
    try:
        with open ( 'report_number.txt' , 'r' )  as file:
            report_number = int(file.read())
    except(ValueError, FileNotFoundError):
        report_number = 1
    return report_number
